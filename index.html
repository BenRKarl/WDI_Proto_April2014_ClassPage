We are WDI.__proto__

From 21 wide-eyed programming newbies, we've been transformed into junior full-stack web developers over the course of General Assembly's 12-week Web Development Immersive. Now we are ready to be unleashed into the world to build meaningful things.

Front-end

We are fluent in HTML5, CSS3, Javascript, AJAX as well as extensions and libraries like SASS, Skeleton, Bootstrap, jQuery, Underscore.js, Backbone.js.

Back-end

We specialize in Ruby on Rails applications with a PostgreSQL database but have also dabbled in Sinatra, Node.js and MongoDB.

Dev tools

WDI graduates are well-versed in Git and Bash. Our applications are tested with RSpec, Jasmine and deployed to Heroku, among other cloud-based hosting.

And more

We are equipped with pair-programming and Object Oriented Programming best practices, as well as the ability to identify and tackle problems.
